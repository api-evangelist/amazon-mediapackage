# Amazon MediaPackage (amazon-mediapackage)
AWS Elemental MediaPackage reliably prepares and protects your video for delivery over the internet. From a single video input, MediaPackage creates video streams formatted to play on connected TVs, mobile phones, computers, tablets, and game consoles using just-in-time packaging.

### AWS Elemental MediaPackage API
